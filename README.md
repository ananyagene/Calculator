# Simple Calculator in Python

## About the Project

This is a simple calculator made using Python. It allows the user to enter two numbers and choose an arithmetic operator. Based on the operator entered, the program performs the calculation and displays the result.

This project was created to practice basic Python concepts such as user input, conditional statements, arithmetic operations, and handling simple errors.

## Features

Takes two numbers as input from the user.
Supports the following operations:

  * Addition (`+`)
  * Subtraction (`-`)
  * Multiplication (`*`)
  * Division (`/`)
Prevents division by zero by displaying an appropriate message.
Easy to understand and beginner-friendly.

## How to Run

1. Open the Python file in any Python IDE or Google Colab.
2. Run the program.
3. Enter the first number.
4. Enter the second number.
5. Enter an operator (`+`, `-`, `*`, or `/`).
6. The program will display the result of the calculation.

## Example

```
Enter first number: 25
Enter second number: 5
Enter an operator: /

divide: 5.0
```

## Python Concepts Used

* `input()` for taking user input
* `float()` for converting input into numbers
* `if`, `elif`, and `else` statements for decision-making
* Arithmetic operators (`+`, `-`, `*`, `/`)
* Basic error handling for division by zero

## Possible Improvements

In the future, this calculator can be enhanced by:

* Adding more operations like modulus (`%`), exponent (`**`), and floor division (`//`)
* Allowing users to perform multiple calculations without restarting the program
* Displaying a clear message when an invalid operator is entered

## Conclusion

This project is a good starting point for anyone learning Python. It demonstrates how user input, conditional statements, and arithmetic operations work together to build a simple and interactive application.
